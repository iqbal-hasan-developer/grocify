# Grocify

Grocify is a modern, user-friendly grocery shopping web application that allows users to browse and purchase fresh, high-quality products from the comfort of their homes. The platform is designed with a focus on simplicity, convenience, and reliability.

## Features

- **Home Page**: A visually appealing landing page with a hero section, categories, featured products, and testimonials.
- **Category Browsing**: Shop by categories such as Fruits, Vegetables, Dairy, Meat, and Seafood.
- **Product Filtering**: Filter products by category and view detailed product cards with images, titles, and prices.
- **Responsive Design**: Fully responsive layout for seamless usage on desktop, tablet, and mobile devices.
- **Interactive UI**: Smooth animations and transitions for an engaging user experience.
- **Testimonials**: Customer reviews displayed in a carousel format.
- **Discounts**: Highlighted first-order discounts and promotional offers.

## Tech Stack

- **Frontend**: React, React Router, Tailwind CSS
- **Build Tool**: Vite
- **Icons**: React Icons
- **Carousel**: Swiper.js
